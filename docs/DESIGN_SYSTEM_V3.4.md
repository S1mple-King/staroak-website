# 星橡官网设计系统 V3.4

## 1. 设计概念

**Sovereign Orbit / 深穹星轨**。以 StarOak Orbit Mark 的轨道、金属银与战略星点为图形语言，建立国际控股集团的深邃、低奢与秩序感。

## 2. Design Tokens

机器可读版本见 `specs/design-tokens.json`。

### 颜色

- `--bg`: #050914
- `--bg-2`: #0B1220
- `--panel`: rgba(13,21,35,.78)
- `--panel-2`: rgba(18,26,42,.62)
- `--line`: rgba(226,214,188,.18)
- `--gold`: #D5B36E
- `--gold-soft`: #9F8450
- `--silver`: #D9DCE1
- `--silver-soft`: #A8ADB7
- `--text`: #F3EFE7
- `--muted`: #A9B0BC

### 布局

- 内容最大宽度：1180–1280px；
- 页面横向 Padding：桌面 56px、平板 32px、移动 18px；
- Section 纵向间距：桌面 96–140px、移动 64–88px；
- 卡片圆角：20px；大容器：28–44px；
- 12 栏网格，卡片间距 18–24px。

### 字体

不依赖第三方字体 CDN。中文正文使用系统字体；标题可使用系统衬线字体栈。保持中英文混排一致和可访问。

## 3. 视觉 Do / Don't

### Do

- 大留白；
- 细金线和钛银描边；
- 低饱和、低对比层次；
- 原创 SVG 星轨和架构图；
- 深色玻璃卡片；
- 轻量慢动效。

### Don't

- SaaS 蓝；
- 世界地图模板；
- 机器人/芯片/握手素材；
- 霓虹赛博；
- 大面积金色；
- 重型 WebGL/视频首屏。

## 4. 核心组件规格

### Header

- Sticky、半透明、18px blur；
- PC 展示全部导航；
- 移动端真实抽屉菜单；
- CTA：生态合作；
- 滚动后高度可略收缩。

### Hero

- PC 两列 55/45；
- H1 48–86px；
- 右侧轨道视觉 480–560px；
- 移动端单列，视觉在文案后；
- 首屏不使用视频。

### Card

- 最小高度由内容决定，不强行统一过高；
- hover 仅轻微位移、边框高亮和阴影；
- 合规卡片始终可见。

### Forms

- 输入高度 48–52px；
- 真实 label；
- 错误信息位于字段下方；
- 成功状态清晰；
- 禁用状态、加载状态和键盘焦点完整。

## 5. 可访问性

- WCAG AA 作为目标；
- 正文对比度不得因低奢风格而牺牲；
- 语义标题顺序；
- 键盘菜单、Escape 关闭抽屉；
- 图片 alt；
- Reduced Motion；
- 表单错误关联 `aria-describedby`；
- 不只用颜色传达状态。
