# Release Notes V3.2

## 新增

- 新增 `/api/leads` 三类表单统一提交接口；
- 新增 `/api/health` 健康检查接口；
- 新增 `/api/revalidate` CMS 发布后刷新接口；
- 新增 Strapi 六类内容模型 schema 草案；
- 新增隐私政策 `/privacy` 与免责声明 `/disclaimer` 页面；
- 表单增加真实 POST、状态反馈、honeypot、防垃圾、隐私授权；
- Footer 增加隐私政策和免责声明入口；
- Sitemap 增加隐私政策和免责声明页面。

## 保持不变

- 网站一级导航仍为首页、关于星橡、AI引擎、产业布局、星橡智库、生态合作、联系我们；
- 沐洋智联、星藤智能科技、企业AI效能平台、APEX 作为内容抓手，不新增一级导航；
- `/ai-core` 继续兼容跳转 `/ai-engine`；
- 官网继续采用深穹星轨、钛银、微光金、低奢控股集团视觉方向。
