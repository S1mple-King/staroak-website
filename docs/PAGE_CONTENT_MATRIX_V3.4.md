## 3. AI引擎 `/ai-engine`

### Hero

- H1：AI 引擎：星橡的核心智能能力

### Hero

- H1：产业布局：当前抓手与未来空间

### 平台卡片

#### 沐洋智联

- Eyebrow：AI Global Growth
- Relation：星橡参股联合打造 / 产业服务抓手
- Summary：面向企业全球市场拓展，整合市场洞察、内容生成、渠道触达、线索管理、AI获客与增长运营能力。
- Points：全球增长服务 / AI获客与线索管理 / 内容与渠道触达 / 增长运营闭环。

#### 星藤智能科技 × 爬山虎

- Eyebrow：AI Acquisition & Operation
- Relation：AI智能获客主要内容
- Summary：围绕全域智能获客与运营，整合 AI客服、客群会话、会话洞察、意向客户识别、热词分析与私域转化。
- Points：全域获客 / AI客服与洞察 / 意向识别 / 私域运营。

#### 企业AI技术效能增长平台

- Eyebrow：Enterprise AI Efficiency
- Relation：企业AI效能增长方案
- Summary：以统一AI工作入口、模型与Token治理、研发场景智能体、效能看板和资产沉淀，帮助技术组织形成AI生产力系统。
- Points：统一AI工作台 / 模型与Token治理 / 研发Agent / 效能与资产库。

#### APEX

- Eyebrow：AI Quant Trading
- Relation：AI量化交易平台
- Summary：全球多市场AI智能交易中枢系统，围绕AI信号工厂、多市场执行中台、统一风控与审计能力进行系统化建设。
- Points：AI信号工厂 / 多市场执行 / 三层风控 / 全链路审计。
