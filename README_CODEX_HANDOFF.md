# StarOak Codex Development Handoff V3.4

这是可直接交给 Codex 的开发包，包含：

- 当前 Next.js 官网工程；
- `AGENTS.md` 代理工作规则；
- Master PRD 与页面、设计、CMS/API、验收规格；
- 机器可读 JSON 规格；
- 已确认视觉参考；
- Docker、Nginx 和部署文档。

## 使用方式

1. 将整个目录交给 Codex 或放入 Git 仓库；
2. 向 Codex 粘贴 `CODEX_START_PROMPT.md` 内容；
3. 要求先生成 `docs/PRD_GAP_REPORT.md`，再编码；
4. 以 `docs/STAROAK_WEBSITE_MASTER_PRD_V3.4.md` 为最终事实源。

## 注意

视觉参考中有早期旧名称，仅参考视觉质感，文案以 Master PRD 为准。
