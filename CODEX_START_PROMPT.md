# 给 Codex 的启动提示词

请以本仓库为开发起点，先读取根目录 `AGENTS.md` 和 `docs/STAROAK_WEBSITE_MASTER_PRD_V3.4.md`，将 Master PRD 视为唯一产品事实源。

第一步不要重写项目，请：

1. 安装依赖并运行 typecheck、lint、build；
2. 审计当前工程与 PRD 的差异，生成 `docs/PRD_GAP_REPORT.md`；
3. 优先完成 Phase 1 和 Phase 2：
   - 修复构建问题；
   - 实现移动端菜单；
   - 补齐首页“一核多翼、一院一网”和价值创造飞轮；
   - 统一设计 Token、响应式和可访问性；
   - 确保 AI引擎、产业布局和四个抓手的最终文案正确；
4. 运行测试并生成本地预览；
5. 最后输出：改动摘要、测试结果、未决事项和下一阶段建议。

不得编造业务数据、牌照、收益、案例或合作关系；不得改变已确认的导航、品牌定位和视觉方向。
